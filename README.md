# Persona Cards

SillyTavern의 페르소나를 하나씩 저장하고 불러올 수 있는 확장프로그램입니다.

전체 페르소나를 한꺼번에 백업하는 대신, 원하는 페르소나만 PNG 또는 JSON 파일로 내보내고 다른 SillyTavern 환경에서 다시 가져올 수 있습니다.

## 주요 기능

- 페르소나를 PNG 또는 JSON 파일로 개별 저장
- 저장한 페르소나 파일 미리보기 및 가져오기
- PNG 저장 시 페르소나 이미지와 정보 포함
- JSON 저장 시 페르소나 정보만 포함
- 연결된 캐릭터·그룹 및 로어북 정보 선택적 포함
- 동일한 페르소나가 있을 때 덮어쓰기 또는 복사본 생성

## 버튼 설명

### 페르소나 가져오기

페르소나 관리 화면 상단의 백업·복원 버튼 옆에 있는 **ID 카드 아이콘**입니다.

PNG 또는 JSON으로 저장한 페르소나 파일을 선택해 불러옵니다. 가져오기 전에 이름, 설명, 로어북 및 연결 정보를 미리 확인할 수 있습니다.

### 페르소나 내보내기

`Current Persona` 영역의 이름 변경·복제·삭제 버튼과 함께 표시되는 **파일 내보내기 아이콘**입니다.

현재 선택한 페르소나를 다음 형식으로 저장할 수 있습니다.

- **PNG로 저장:** 페르소나 이미지와 정보를 하나의 PNG 파일에 저장합니다.
- **JSON로 저장:** 이미지를 제외한 페르소나 정보만 JSON 파일에 저장합니다.

내보내기 창에서 캐릭터 및 그룹 연결 정보 포함 여부를 선택할 수 있습니다.
